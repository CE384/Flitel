## Setup 


```Bash
git clone https://github.com/CE384/Flitel.git
cd Flitel/flitel
python3 -m venv env
./env/bin/activate
pip install -r requirements.txt
cp database.ini.example database.ini
./run.sh   
```
- Fill your postgres database credential in Flitel/flitel/database.ini

### Windows

```Bash
py -3 -m venv env
pip install -r .\requirements.txt
env\Scripts\activate                              
\run.sh    
```
